## 类的定义
* 类的定义就是通过使用class关键字来定义一个类，类的名称通常采用驼峰命名法。
* 类的定义格式如下：
```python
class 类名:
    类体
```
* 类的实例化：通过类创建对象的过程叫做类的实例化，类的实例化就是创建一个类的对象
