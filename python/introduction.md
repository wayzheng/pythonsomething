#### 计算机基础
* 计算机组成：CPU 内存 外存 接口 外设
* CPU 就是中央处理器单元，负责运算，运算速度很快，但是只能做运算，不能做输入输出。
* 内存就是用来存储数据的 
* 外存也是用来存储数据的 但是是在硬盘里，但是速度慢。
* 接口就是用来输入和输出了 用来和外界设备进行交互
* 计算机架构就是 CPU+内存+外存+接口
* 冯诺依曼架构提出程序和数据都是可以保存在存储器中
* 环境搭建 推荐使用pycharm 使用vscode也可以 不推荐使用idle来进行学习
* 编程语言的历史
* 解释型语言和编译型语言
* 解释型语言
    * 代码是一行一行的执行的 所以相对于编译型语言来说速度会慢一些
    * 需要通过解释器来执行代码
* 编译型语言
  * 代码就是一次性进行编译之后再进行执行的
  * 运行速度非常快
  * 需要通过使用编译器来进行编译
  * 比如说C语言就是编译型语言
  * 代码是一次性编译的
* 但是java是一种特殊的语言 他是一种解释型语言和编译型语言的结合体
* java代码是一行一行的执行的 但是java代码在执行之前会先进行编译

#### 变量
* 在python中定义一个变量是非常简单的
* 变量名的命名规则
  * 变量名只能包含字母 数字 下划线
  * 变量名不能以数字开头
  * 变量名不能包含空格
  * 变量名不能包含特殊字符
  * 变量名不能是python的关键字
* python关键字
    * and as assert break class continue def del elif else except False finally for from global if import in is lambda None nonlocal not or pass raise return True try while with yield
* 由于python是动态语言 所以变量的类型是可以改变的
* 可以试一下下面的代码
```python
a = 1
print(a, type(a))
a = "hello world"
print(a, type(a))
a = [1, 3, 5]
print(a, type(a))
```
* 也就是说变量是可以不用进行声明的 但是后续来构建函数的时候 编写docstring的时候最好还是标识数据的类型
#### 注释
* python中的注释有两种
* 单行注释
  * 使用#来进行注释
* 多行注释
  * 使用三个单引号或者三个双引号来进行注释
  ```python
  """hello world
  这是一个多行的注释"""
  ```
* 其实还有一种文档注释 推荐使用文档注释来对自己的函数和类进行注释说明
* 文档注释
  * 使用三个双引号来进行注释
  * 但是文档注释是用来生成文档的
  * 一般是用来生成函数的文档
  ```python
  def add(a, b):
    """
    这是一个加法函数
    :param a: int
    :param b: int
    :return: int
    """
    return a + b
  ```
#### 条件语句
* python中的条件语句有if elif else
* if语句
  * if语句是用来判断一个条件是否成立的
  * 如果条件成立就执行if语句下面的代码
  * 如果条件不成立就不执行if语句下面的代码
  * 语法
  ```python
  if 条件:
    代码块
  ```
* elif语句
* elif语句是用来判断多个条件的
* 如果if语句不成立 就会判断elif语句
* 如果elif语句成立 就会执行elif语句下面的代码
* 语法
```python
if 条件:
  代码块
elif 条件:
  代码块
```
* else语句
* else语句是用来判断所有的条件都不成立的时候执行的
* 语法
```python
if 条件:
  代码块
elif 条件:
  代码块
else:
  代码块
```

* while循环是用来判断一个条件是否成立的

### 进制和编码
#### 进制的介绍
* 二进制 - 计算机中最基础的一种进制 由0和1组成
* 八进制 - 由0-7组成 就是相当于是由三位二进制数组成
* 十进制 - 由0-9组成 
* 十六进制 - 由0-9 a-f组成 就是相当于是由四位二进制数组成
#### 编码的介绍
* 编码就是encode 译码就是decode
* 常见的编码规则有ASCII码和Unicode码
* ASCII码
  * ASCII码是由美国人发明的 用来表示英文字符的
  * ASCII码是一个字节 也就是8位
  * 一个字节可以表示256个字符
  * 但是ASCII码只能表示英文字符
  * 比较常见的对应关系就是
  * A -> 65 a -> 97 0 -> 48
* Unicode码
  * Unicode码是由国际组织发明的 用来表示全球所有的字符
  * Unicode码是两个字节 也就是16位
  * 一个字节可以表示65536个字符
  * 但是Unicode码是一个很大的码表
  * 为了节省空间 有人发明了UTF-8码
* UTF-8是现在比较常见的编码规则 兼容中文和英文 比如网页中的 <meta charset="utf-8" /> 就是使用的UTF-8编码 从而有效地避免了乱码的问题

### 前言
* **程序 = 算法 + 数据结构**
* 算法的简介 algorithm
* 算法就是解决问题的准确而完整的描述 是一系列解决问题的清晰指令，算法代表着用系统的方法来描述解决问题的策略机制， 也就是说， 能够对一定规范的数据，在有限的时间和空间内，获得所要求的结果，任何代码片段都是可以看做是一个算法

### 大O表示法
* 大O表示法使用大写字母O 可以认为其含义就是"order of" 就是大约是的意思
* 大O表示法可以告诉我们算法的快慢
* 大O比较的就是操作数 它指出了算法运行时间的增长趋势
* O(n)括号里面的n就是操作数
* 注意 大O表示法表示的就是一个算法在最坏的情况下的运行时间
* 常见的大O运行时间
    * O(log n) 也叫做对数时间 二分查找
    * O(n) 也叫做线型时间 简单查找
    * O(n * log n) 快速排序 一种速度较快的排序算法
    * O(n ^ 2) 选择排序 一种速度很慢的一种排序算法
    * O(n!) 旅行商问题的解决方案 一种非常慢的一种算法
* 其中O(1)是优秀 O(log n)就是良好 O(n)就是可以 O(n ^ 2)就是很差了
* 算法的速度指的是操作数的增速 而非时间
* 讨论算法素的说的是随着输入的增加 其运行的时间将会以什么样的速度增加
* 使用大O表示法表示算法的运行时间
* 随着元素的增加 快算法比慢算法增加的速度是指数级的 比如O(log n)和O(n)
